# Week4tut
 
Command used for this project.
 
## Angular
 
### 1. Create a new angular app
 
```sh
ng new week4tut
```
 
### 2. Add bootstrap to my application (npm)
 
```sh
npm install bootstrap --save
```
 
### 3. Create page components
 
```sh
ng generate component login
ng generate component account
```
 
### 4. Start application
 
```sh
ng serve
```
